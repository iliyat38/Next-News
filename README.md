# 📰 Next News — Modern News Platform Built with Next.js

A modern and fully responsive news application built with **Next.js App Router**.

Built as part of an advanced Next.js learning project focused on mastering the App Router and modern routing patterns.

As my first Next.js project, the focus was not only on building a functional news platform but also on gaining hands-on experience with advanced routing patterns, layouts, API handling, and modern React development practices.

---

## 🚀 Key Features

* 📰 **Latest News Feed** — Browse news articles through a clean and responsive interface.

* 📅 **News Archive Filtering** — Filter news articles by year and month through dynamic archive routes.

* 🔍 **Dynamic News Pages** — Each article is rendered through dynamic route segments for scalable content management.

* 🧭 **Advanced App Router Architecture** — Built entirely with the Next.js 16 App Router.

* ⚡ **Parallel Routes** — Render multiple route segments simultaneously for improved user experience.

* 🪟 **Intercepting Routes & Modal Navigation** — Open news details inside modal overlays while preserving the current page context.

* 📂 **Route Groups** — Organized route structure using route groups for better maintainability.

* 🎯 **Dynamic Routes** — Flexible content rendering through dynamic segments.

* 🪝 **Catch-All & Optional Catch-All Routes** — Advanced URL handling for archive and nested content pages.

* 🔄 **Axios Integration** — Centralized API communication layer.

* 🛡️ **Axios Interceptors** — Reusable request and response interceptors for cleaner data handling.

* ⏳ **Loading UI** — Dedicated loading states using Next.js loading conventions.

* ❌ **Error Boundaries** — Robust error handling with fallback UI.

* 🎨 **Responsive Design** — Optimized for desktop, tablet, and mobile devices.

* 🖼️ **Next.js Image Optimization** — Utilized the `next/image` component for automatic image optimization, lazy loading, and improved performance.

* 🚀 **Performance-Focused Rendering** — Leveraged built-in Next.js optimizations for faster page loads and better user experience.

* 🔗 **URL-Driven Navigation** — News archives and article pages are fully shareable through dynamic URL structures.
